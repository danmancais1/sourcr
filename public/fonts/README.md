# Aspecta font files

Place Aspecta Sans Serif font files here for the typography system:

- `Aspecta-Regular.woff2` (weight 400)
- `Aspecta-Medium.woff2` (weight 500)
- `Aspecta-SemiBold.woff2` (weight 600)
- `Aspecta-Bold.woff2` (weight 700)

If the font files are not present, Inter (loaded via next/font) will be used as fallback.
